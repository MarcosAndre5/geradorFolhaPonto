# Gerador de Folha Ponto
Projeto em Python para geração de folha de pontos de funcionários.

### Pré-requisitos
* Python 3
* Pandas
    ```
    pip install pandas
    ```

### Gerando o arquivo executável
* Pyinstaller
    ```
    pip install pyinstaller
    ```
    
    ```
    pyinstaller <nome_arquivo.py>
    ```
